# Okay-Garmin
Based on: https://www.youtube.com/embed/QeX0wYlnUt0

This project allows you to create a clip in Medal by saying the voice command:

> **"Okay Garmin, Video speichern"**

When the phrase is detected, Medal will automatically save the current video clip.

---

## 🛠️ How It Works

1. The program listens for the voice phrase.
2. Once detected, it triggers Medal’s clip saving action.
3. The video gets saved instantly.
